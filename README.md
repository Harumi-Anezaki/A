# Cross-Platform Todo App

A cross-platform Todo application built with Flutter, targeting both Android and Web.

## Features
- **Modern UI**: Styled with Material 3 design principles for a vibrant and responsive look.
- **Task Management**: Add new tasks, toggle their completion status, and delete them.
- **Data Persistence**: Uses `shared_preferences` to save tasks locally. Your data persists even after restarting the app or refreshing the browser.

## Getting Started

This project is a Flutter application. To run this project locally, ensure you have Flutter installed.

### Run on Web
To launch the app in Chrome:
```bash
flutter run -d chrome
```

### Run on Android
To launch the app on an Android emulator or connected device:
```bash
flutter run -d android
```
